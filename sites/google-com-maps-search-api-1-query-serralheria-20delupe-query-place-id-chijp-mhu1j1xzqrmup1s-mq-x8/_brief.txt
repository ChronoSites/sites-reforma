  soluções completas em estruturas metálicas, portões, grades, corrimãos,
  coberturas para galpões e peças de design sob medida. Sua nota de 4.9
  com 63 avaliações no Google Maps comprova a excelência e a confiança
  conquistada junto a clientes residenciais e comerciais.

  2. CONCEITO VISUAL — "INDUSTRIAL PREMIUM"
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  O conceito visual mistura a robustez do aço com a sofisticação do
  design contemporâneo. Tons escuros de carvão e grafite (remetendo ao
  metal) são contrastados por laranjas e âmbares quentes (remetendo a
  faíscas e solda), criando uma identidade que transmite força, perícia
  e modernidade.

  5. LAYOUT & GRID
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  • Approach ........... Mobile-First, Single Page com seções
  • Max-width .......... 1200px (container centralizado)
  • Grid ............... CSS Grid + Flexbox
  • Gutter ............. 24px (desktop) / 16px (mobile)
  • Padding seções ..... 100px vertical (desktop) / 60px (mobile)
  • Seções alternadas .. Background escuro / escuro variante
                         criando ritmo visual

  Breakpoints:
    Mobile:  < 768px
    Tablet:  768px — 1024px
    Desktop: > 1024px

  6. ESTILO VISUAL
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  • Tom geral .......... Dark mode industrial premium
  • Border radius ...... 12px (cards) / 8px (botões) / 50% (avatares)
  • Sombras ............ 0 8px 32px rgba(0,0,0,0.3) — profundidade dramática
  • Hover effects ...... Scale 1.03 + sombra amplificada
  • Transições ......... 0.3s cubic-bezier(0.4, 0, 0.2, 1) — suave e moderna
  • Gradientes ......... Sutis, de #1A1A2E → #16213E nas seções
  • Overlay em imagens . rgba(26,26,46,0.6) com texto por cima
  • Linha accent ....... Barras de 4px #E67E22 sob títulos de seção
  • Glassmorphism ...... Cards com backdrop-filter: blur(10px)
                         e borda semi-transparente
  • Animações scroll ... Fade-in + slide-up ao entrar no viewport
                         usando Intersection Observer

  CARDS DE PORTFÓLIO
  ├─ Imagem com aspect-ratio 4:3, object-fit cover
  ├─ Overlay gradient no hover com título da categoria

  CARDS DE SERVIÇO
  ├─ Ícone SVG grande (64px) em cor accent
  ├─ Título + descrição curta
  ├─ Borda left 4px #E67E22
  └─ Hover: translateY(-8px) + sombra

  NAVBAR
  ├─ Fixa no topo, backdrop-filter: blur(12px)
  ├─ Logo à esquerda, links centralizados, CTA à direita
  ├─ Hamburger em mobile com menu fullscreen
  └─ Scroll-spy para destacar seção ativa

  HERO SECTION

  ├─ Overlay escuro com gradiente
  ├─ Título impactante + subtítulo + 2 CTAs
  ├─ Scroll indicator animado (chevron pulsante)
  └─ Parallax sutil no background
